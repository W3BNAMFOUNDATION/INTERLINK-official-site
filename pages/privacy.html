Privacy Policy

Your privacy is critically important to us.

1. Overview

INTERLINK is designed with sovereign local processing. All data stays inside your environment unless YOU explicitly connect external APIs.

2. Data Collection

INTERLINK does not collect or transmit data to external servers. No analytics, no telemetry, no hidden logging.

3. External Services

If you choose to integrate external LLMs (DeepSeek, Groq, OpenAI, etc.), you remain fully responsible for those connections.

4. Local Processing

  • All ingestion is processed locally
  • Memory is stored locally
  • No external persistence unless configured by the user

5. Security

We recommend secure API key storage, encrypted environment variables, and responsible operational security practices.

6. User Rights

You control all data processed by INTERLINK. You may delete, modify, or export your data at any time.

7. Updates

Privacy changes will be published directly on this page. Continued use of the software indicates acceptance of these terms.
