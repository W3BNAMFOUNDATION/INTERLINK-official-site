Workspace quickstart

  1. Install the desktop app and sign in with your workspace key.
  2. Create your first project and select your preferred LLM stack (online/offline).
  3. Connect storage: choose local embeddings or your enterprise vector database.
  4. Import a PDF, URL, or repository into memory and start chatting with the context.

APIs

Call INTERLINK™ 2.0 from any stack. REST and WebSocket endpoints are available once the desktop service is running.

EndpointDescription
/api/v2/agentsCreate, list, and manage agent profiles.
/api/v2/memory/searchQuery embeddings with hybrid semantic/keyword filters.
/api/v2/automation/runTrigger scripted automations with signed scopes.
Add your workspace token to `Authorization: Bearer <token>` for every request.

Security controls

  • ✓ Per-project network and filesystem scopes
  • ✓ Agent sandboxing with user confirmation gates
  • ✓ Audit log export for compliance teams

Extensibility

Drop in your own models, memory backends, and automation skills. Plug-ins are isolated containers that inherit workspace permissions.
