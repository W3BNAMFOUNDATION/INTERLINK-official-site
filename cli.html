üß† What it can do

Send AI messages

Chat with DeepSeek, Ollama, Groq, OpenAI, or any routed model with the same prompt format.

python interlink.py chat deepseek "Explain sovereign AI."
python interlink.py chat ollama  "Hello world"
python interlink.py chat groq    "Summarize this text..."
python interlink.py chat openai  "What is INTERLINK?"

Ingest anything

Run the safe ingest pipeline for files, URLs, or inline text with automatic cleaning and embedding.

python interlink.py ingest file ./pdfs/report.pdf
python interlink.py ingest url "https://example.com"
python interlink.py ingest text "My notes go here..."

Search memory

Query vector memory and return semantic chunks with embeddings and relevance scores.

python interlink.py memory "quantum encryption"

Run agents

Trigger summarization, classification, and any automation-ready agent profile.

python interlink.py agent summarize "workspace/ingest/*.pdf"
python interlink.py agent classify  "workspace/uploads/*.txt"

Filesystem commands

Use the FS-Core sandbox to list, read, clean, or reset workspace directories.

python interlink.py fs list memory/
python interlink.py fs read memory/index.json
python interlink.py fs delete tmp/*
python interlink.py fs reset

Automation + debug

Execute tasks and inspect every service from the terminal.

python interlink.py task run auto_pdf_pipeline
python interlink.py debug services
python interlink.py debug workspace
python interlink.py debug memory
python interlink.py debug ingest
